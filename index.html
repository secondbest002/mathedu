Manual Operasional

Bilangan Bulat Positif (+)

Nilai di atas nol. Analogi: Uang/Aset atau langkah Maju. Pada sistem ini, tanda (+) berarti unit menghadap ke KANAN.

Bilangan Bulat Negatif (-)

Nilai di bawah nol. Analogi: Hutang atau langkah Mundur. Pada sistem ini, tanda (-) berarti unit menghadap ke KIRI.

Kenapa (-a) x (-b) = Positif?

Bayangkan unit menghadap KIRI (karena -a). Operasi kali negatif berarti "Lakukan pergerakan mundur". Jika kamu menghadap kiri lalu berjalan mundur, kamu sebenarnya bergerak ke arah KANAN (Positif).

NEGATIF (KIRI) POSITIF (KANAN) 0

Garis Bilangan: Representasi visual urutan nilai integer.

TANDA OPERASI (+) Artinya: MAJU
TANDA OPERASI (-) Artinya: MUNDUR
BILANGAN POSITIF Artinya: HADAP KANAN
BILANGAN NEGATIF Artinya: HADAP KIRI
Simulasi Kasus: -2 + 2
1. Bilangan -2 (Negatif) -> Hadap KIRI, Maju 2 langkah.
2. Operasi (+) -> Siap MAJU.
3. Bilangan 2 (Positif) -> Hadap KANAN.
4. Eksekusi: Maju 2 langkah.
Hasil: Kembali ke 0.

Pada perkalian, bilangan pertama adalah Pengali (Menentukan arah hadap awal), bilangan kedua adalah Yang Dikalikan (Besar lompatan).

  • Pengali Positif: Hadap KANAN.
  • Pengali Negatif: Hadap KIRI.
  • Dikali Positif: Melompat MAJU.
  • Dikali Negatif: Melompat MUNDUR.
